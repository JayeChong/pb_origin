**一个数N的二进制和（N-1）的二进制做与运算的到的结果总比N少一个1**
根据这个原理我们可以让 N 和 N-1 做与运算得到新的值 M（二进制M 1 的个数比 N 中 1的个数少一个） 再和 M-1 做与运算，重复直到得到的值为0。（最终所有的1都会被与掉）

```javascript
// 计算一个数的二进制中1的个数
const n = (num) => {
    let count = 0;
    while (num) {
        num &= num - 1;
        count ++;
    }
    return count;
}
```
